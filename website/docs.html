Default Parameter Configuration

Default DNA2Graph parameters were optimized and validated for the following acquisition setup:

  • MAPS Set Up: 17436nm
  • Microscope: Talos F200C G2
  • TEM mode: bright field
  • Bright field mode: microprobe
  • Condenser 2 aperture (microns): 150 um
  • Objective aperture (microns): 30 um
  • Camera: Ceta-S
  • Tile set: 4x4
  • Overlap: 10%
  • Resolution pixels: 4096x4096
  • Magnification: 8500X
NOTE: Different acquisition setups may require parameter tuning.
